# Images Directory

Place your images here. Recommended structure:

```
images/
├── hero/
│   └── hero-bg.jpg (or .mp4 for video)
├── products/
│   ├── veste-memoire-1.jpg
│   ├── veste-memoire-2.jpg
│   ├── tshirt-trace-1.jpg
│   └── ...
├── collection/
│   └── souvenir-bleu-story.jpg
├── about/
│   ├── about-hero.jpg
│   └── philosophy.jpg
└── textures/
    └── grain.png (optional texture overlay)
```

## Image guidelines:
- Use high-quality images (minimum 1920px width for hero)
- Optimize images for web (WebP format recommended)
- Product images: 3:4 aspect ratio recommended
- Hero images: 16:9 or wider
