# Fonts Directory

Download fonts from Google Fonts and place them here:

## Required fonts:

### Inter (sans-serif - body text)
- Inter-Light.woff2
- Inter-Regular.woff2
- Inter-Medium.woff2

### Cormorant Garamond (serif - display/titles)
- CormorantGaramond-Light.woff2
- CormorantGaramond-Regular.woff2
- CormorantGaramond-LightItalic.woff2
- CormorantGaramond-Italic.woff2

### Playfair Display (serif - accents)
- PlayfairDisplay-Regular.woff2
- PlayfairDisplay-Italic.woff2

## Download links:
- Inter: https://fonts.google.com/specimen/Inter
- Cormorant Garamond: https://fonts.google.com/specimen/Cormorant+Garamond
- Playfair Display: https://fonts.google.com/specimen/Playfair+Display
